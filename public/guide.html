🍄 서비스 소개

메이플스토리 Nexon Open API를 통해 거울 · 달라 길드원 데이터를 매일 자동 수집하고, 성장 현황을 시각화해 제공하는 웹 대시보드입니다.

📊
주요 제공 정보
  • 일별 / 주간 / 월간 경험치 · 전투력 성장 랭킹
  • 길드원 전체 목록 (직업 · 레벨 · 전투력)
  • 개인 프로필 페이지 (성장 추이 차트)
  • 가입 · 탈퇴 디스코드 실시간 알림
  • 장기 미접속 알림 · 주간 인원 리포트
💡 데이터는 매일 자정 이후 전날 기준으로 수집됩니다. 오늘 성장은 익일 반영됩니다.

📈 성장 대시보드

탭 전환

상단의 일별 / 주간 / 월간 탭으로 기간을 선택합니다.

탭기준설명
일별어제 하루전날 대비 경험치·전투력 변화량
주간목요일 00시 리셋이번 주 누적 성장 (메이플 주간퀘 기준)
월간1일 ~ 말일이번 달 1일 이후 누적 성장

포디움 & 랭킹

경험치 · 전투력 각각 상위 3명 포디움 + 전체 순위표 제공합니다.

  • 현재 스펙 / 변화량 토글로 전환 가능
  • 레벨업 구간의 경험치는 레벨 테이블 기반으로 정확하게 계산
  • 수집된 스냅샷이 없는 기간은 랭킹에서 제외됩니다

👥 길드원 현황

목록 필터

  • 닉네임 검색: 실시간 필터링
  • 직업 필터: 전체 / 개별 직업 선택
  • 레벨 구간: 260 ~ 300 범위 직접 입력

개인 프로필

캐릭터 이름 클릭 시 개인 페이지로 이동합니다.

  • 경험치 · 전투력 성장 추이 라인 차트
  • 최근 스냅샷 기반 스탯 표시
  • MapleScout 외부 링크 연동
💡 프로필 차트는 수집된 스냅샷 개수가 많을수록 정확합니다 (최소 2일치 필요).

🛠️ 관리자 페이지

접근 방법

URL: /admin/{관리자키}

관리자키는 환경변수 ADMIN_SECRET에 설정된 값입니다.

주요 기능

길드원 정보 관리
  • 직위 변경 (마스터 / 부마스터 / 길드원 / 새내기)
  • 실명 · 주 캐릭터 · 비고 · 비활성 사유 입력
  • 거울 ↔ 달라 길드 이전 처리
  • 가입 / 탈퇴 / 데이터 초기화
이탈 감지 알림
  • 5분 배치가 API와 실제 멤버 목록을 비교해 이탈 감지 표시
  • 이탈감지 표시된 멤버를 확인 후 직접 탈퇴 처리
  • 자동 삭제 없음 — 운영진이 직접 확인 후 처리

상태 배지 설명

배지설명
활성 현재 길드 소속 멤버
이탈감지 API에서 더 이상 확인되지 않음 — 탈퇴 또는 이름 변경 가능성
새내기 최근 가입 · 아직 정보 미입력
탈퇴 탈퇴 처리 완료 (목록 표시 유지, 데이터 보존)

📡 디스코드 자동 알림

알림 종류주기내용
가입 감지 5분마다 새 길드원 가입 알림 (거울 · 달라)
탈퇴 감지 5분마다 길드원 이탈 알림 (거울 · 달라)
장기 미접속 매일 07:00 15일 이상 미접속 길드원 목록
주간 리포트 매주 목 09:00 인원 변동 · 성장 하이라이트 요약
⚠️ 디스코드 알림은 DISCORD_WEBHOOK_URL 환경변수 설정 시에만 발송됩니다.

⚙️ 자동 배치 스케줄

GitHub Actions + cron-job.org 외부 트리거로 정확한 주기로 실행됩니다.

배치주기KST역할
길드원 모니터링 5분마다 - 거울/달라 가입·탈퇴 감지 + 디스코드 알림
오늘 실시간 수집 30분마다 - 거울 오늘 스냅샷 수집 (당일 확인용)
전날 결산 수집 매일 00:05 거울 전날 최종 스냅샷 (실패 시 매시간 :35에 재시도)
장기 미접속 알림 매일 07:00 거울/달라 장기 미접속 디스코드 알림
주간 리포트 매주 목 09:00 거울/달라 주간 리포트 디스코드 발송
💡 cron-job.org 설정 방법: 배치 스케줄 설정 가이드

❓ 자주 묻는 질문

오늘 성장이 왜 안 보여요?

데이터는 전날 기준으로 수집됩니다. 오늘 한 활동은 내일 자정 이후 반영됩니다.

랭킹에 내 캐릭터가 없어요.

해당 기간의 스냅샷이 없으면 랭킹에서 제외됩니다. 신규 가입 후 2일치 데이터가 수집되면 자동으로 표시됩니다.

이름 변경 후 추적이 안 돼요.

Nexon API는 OCID(고유 캐릭터 ID) 기반으로 조회합니다. 이름 변경 시에도 동일 OCID로 추적됩니다. 단, 캐시 갱신에 최대 하루 소요될 수 있습니다.

배치가 실행됐는데 디스코드 알림이 안 와요.

GitHub Secrets에 DISCORD_WEBHOOK_URL이 등록되어 있는지 확인해 주세요.

관리자 페이지 접근이 안 돼요.

URL이 /admin/{ADMIN_SECRET값} 형식인지 확인해 주세요. 대소문자를 구분합니다.
